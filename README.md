# ☕ Starbucks Analysis Dashboard | Power BI

An interactive **Power BI dashboard** built to analyze Starbucks beverage data. This dashboard provides insights into beverage nutrition, caffeine levels, calories, sugar content, beverage categories, and Starbucks' global presence using interactive visualizations and filters.

## 🚀 Features

- 📊 KPI Cards
  - Total Beverages
  - Average Sugar
  - Average Calories
  - Average Caffeine

- 📈 Interactive Visualizations
  - Average Calories by Beverage Category
  - Average Caffeine by Category
  - Beverage Category Distribution
  - Starbucks Global Presence (Map)

- 🎛 Interactive Filters
  - Protein Range Slider
  - Beverage Preparation Slicer

- 🌍 Geographic Analysis
  - Starbucks store locations across different countries

---

## 📌 Key Insights

- Coffee contains the highest average caffeine among all beverage categories.
- Beverage categories vary significantly in calories and sugar content.
- Beverage Preparation and Protein Range filters dynamically update all visuals.
- The dashboard enables quick comparison of nutritional values across beverages.
